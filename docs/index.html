System Architecture

Phone (Discord)
  → Discord WebSocket Gateway
  → PicoClaw (static Go binary, arm64)
  → Claude / Gemini / NVIDIA NIM cloud APIs (via WhalechipAI proxy)
  → led_ctrl (C, GPIO chardev API)
  → /dev/gpiochip2 (Linux GPIO chardev)
  → AXI GPIO IP (0xA0000000)
  → FPGA PL LEDs (active-low)

Hardware: ALINX AXU15EG (XCZU15EG-2FFVB1156I)
  PS:  Quad-core Cortex-A53, 4 GB DDR4, Linux 6.6.40
  PL:  UltraScale+ fabric, 2 GB DDR4 (MIG PHY), AXI GPIO
  I/O: SD card boot, USB WiFi (RTL8812BU), GbE, UART

Boot: FSBL → PMU FW → ATF → U-Boot → Kernel → OpenWrt procd (hybrid)
